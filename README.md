# Productivity Operating System (POS)

A modern productivity dashboard built with React and Tailwind CSS. The goal of this project is to create an all-in-one productivity platform that helps users manage tasks, stay focused, and improve daily workflow.

## 🚀 Features

### Dashboard

* Live digital clock
* Today's tasks overview
* Task completion tracking
* Spotify music player section
* Clean and modern dark UI

### Todo Management

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Search tasks
* Filter tasks (All / Pending / Completed)
* Local Storage persistence

### Navigation

* Multi-page application using React Router
* Dashboard page
* Todo Management page
* Pomodoro page (In Progress)
